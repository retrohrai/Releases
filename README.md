# RetroHrai! - Android Frontend Launcher for Retro (Not Only) Gaming

A modern, unified frontend launcher for Android that brings all your games together in one beautiful interface. Play games from multiple emulators - retro classics, modern platforms, and everything in between - without juggling between different apps.

## âœ¨ Features

### Core Functionality
- **Unified Game Library** - Access games from multiple emulators in one place
- **Multiple View Modes** - Switch between Carousel, List, and Grid views
- **Smart Game Organization** - Browse by platform with automatic game detection

### Emulator Support
- **Wide Compatibility** - Works with popular emulators across multiple platforms
- **Per-Game Emulator Override** - Choose different emulators for individual games
- **Windows Platform Support** - Launch Windows games via Winlator/GameHub

### Visual Experience
- **Animated Backgrounds** - Dynamic visual effects
- **3D Card Animations** - Optional 3D effects for active game cards
- **Game Card Overlays** - Cover art styled like physical cartridges
- **Custom Launch Animations** - Personalized game launch transitions
- **Dual Screen Support** - Enhanced interface for devices with secondary screens

### Metadata & Scraping
- **Multiple Providers** - IGDB, TheGamesDB, ScreenScraper, and SteamGridDB integration
- **Automatic Media Download** - Scrape cover art, backgrounds, and metadata for your games
- **Multi-Provider Browsing** - Compare results from all providers in tabbed interface

### Customization
- **Custom Backgrounds** - Personalize your launcher with your own wallpapers
- **Platform Display Options** - Toggle platform titles and logos
- **Hidden Games** - Hide games you don't want in your library
- **Auto-Hide UI** - Options to automatically show/hide UI elements
- **Hardware Button Mapping** - Bind physical buttons to control brightness, volume, and launcher actions

## ğŸš€ Installation

1. Download the latest APK from the [Releases](https://github.com/retrohrai/Releases/releases) page
2. Enable "Install from Unknown Sources" in your Android settings if needed
3. Install the APK
4. Launch RetroHrai! - the setup wizard will guide you through adding your game folders and automatically scanning your platforms

> **Security Note:** If Google Play Protect flags the app, you can disable it temporarily. Once the app is available on Google Play Store, this won't be necessary.
